A responsive HTML/CSS website code for an authentic Spanish restaurant menu.

## Description

This project provides a digital menu interface for "Reks Restaurant," featuring classic Spanish dishes including Tapas, Paellas, and Desserts. It includes a visual menu with images and descriptions, as well as an integrated ordering form structure.

## Features

*   **Responsive Grid Layout:** Menu items are automatically arranged in rows of 3 or 4 depending on screen width using CSS Grid.
*   **Categories:** Distinct sections for Tapas, Paellas, and Desserts.
*   **Order Form:** A built-in HTML form allowing users to select quantities for specific dishes and add special requests per category.
*   **Accessible Navigation:** Simple anchor links to jump between the visual menu and the order form.

## Usage

1.  Download the `reks_menu.html` file.
2.  Open the file in any modern web browser (Chrome, Firefox, Safari, Edge).
3.  Scroll to view the dishes or click the navigation links to place an order.

## Technologies Used

*   HTML5
*   CSS3 (Grid Layout)
